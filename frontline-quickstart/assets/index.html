Next steps:

Configure the callbacks to finish the serverless setup. There are 3 areas where we need to add these callbacks.

First, add CRM Callbacks, Outgoing Conversations Callback, and Template Callback to the Frontline Callbacks Configuration Page. Don’t forget to save.

Next, add the Custom Routing Callback URL below to the Frontline Inbound Routing page. Remember to select the Custom Routing checkbox, and don’t forget to click on Save.

Finally, add the Conversation webhooks to the Pre and Post-event Webhook URLs in the Conversations API webhook configuration page. Select the onConversationAdd and onParticipantAdded options under Webhook Filtering and click on Save.

That's it! We have now configured all the callbacks needed. Now lets head back to the Quickstart Guide for step 6.
