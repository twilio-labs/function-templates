Configure the callbacks to finish the serverless setup. There are 3 areas where we need to add these callbacks.

First, add the CRM Callbacks, Outgoing Conversations Callback, and Template Callback below to the Frontline Callbacks Configuration Page. Don’t forget to save.

Next, add the Custom Routing Callback URL below to the Frontline Inbound Routing page. Remember to select the Custom Routing checkbox, and don’t forget to save.

As the final step, add the Conversations Webhook URL below to the Pre and Post-event Webhook URLs in the Conversations API webhook configuration page. Then select the onConversationAdd checkbox under Pre-webhooks. Next, select the onParticipantAdded checkbox under the Post-webhooks. Finally, don't forget to save.
