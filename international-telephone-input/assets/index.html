International Telephone Input

Validate phone numbers with automatic country detection and the Twilio Lookup API

Try it out

Enter a phone number below to validate it. The form uses the intl-tel-input library for formatting and automatic country detection based on your location.

How it works

  1. The form automatically detects your country
  2. As you type, the number is formatted to your country's standard
  3. When you submit, the number is validated using Twilio's Lookup API
  4. You'll see a success or error message with details

Learn more: Read our blog post about international phone input.
